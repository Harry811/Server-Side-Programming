HTML Text Formatting Tags

Tag Example Description
<b> Bold text Makes text bold.
<i> Italic text Makes text italic.
<u> Underlined text Underlines text.
<s> Strikethrough text Adds a strikethrough effect to text.
<mark> Highlighted text Highlights text.
<small> Smaller text Reduces text size.
<sub> H2O Displays text as subscript.
<sup> 2nd place Displays text as superscript.
<strong> Strong emphasis Indicates strong importance or emphasis.
<em> Emphasized text Indicates emphasis, usually rendered in italics.
<blockquote>
This is a blockquote
 Indicates a block of quoted text, usually indented.
<code> Inline code Displays inline code text with a monospace font.
<pre> 
Preformatted text
 Displays preformatted text, preserving spaces and line breaks.
